🏢 Multitenant Rezervační Systém

Příklady vložení iframe pro různé firmy s vlastním company ID

📌 Jak to Funguje?

Každá firma má svůj iframe s vlastním ?companyId=X

Frontend automaticky detekuje company ID a pracuje jen s daty dané firmy.

🔒 Bezpečnost

Uživatelé vidí jen rezervace a údaje své firmy.

Data jsou filtrována na backendu (server-side).

🚀 Nasazení

Jedna aplikace na webu, více URL s různými company ID.

Snadné škálování na desítky či stovky firem.

Klub Pincerna Company ID = 1
<iframe src=".../?companyId=1"></iframe>
Frontend detekuje ?companyId=1 z URL a posílá to všem API requestům. Backend vytváří rezervace v company_id = 1.
Klub Vsetín Company ID = 2
<iframe src=".../?companyId=2"></iframe>
Frontend detekuje ?companyId=2 a posílá to v request body. Backend vytváří rezervace v company_id = 2.

📖 Jak Vložit na Svůj Web

Nahraďte 1 nebo 2 váším company ID:

<iframe
  src="https://tvuj-server.cz/rezervace/?companyId=1"
  width="100%"
  height="800"
  frameborder="0"></iframe>
